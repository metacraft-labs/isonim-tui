# isonim-tui gh-pages

Benchmark baseline store for [isonim-tui](https://github.com/metacraft-labs/isonim-tui)
(milestone M24, Continuous Benchmarking).

This branch is owned by
[`benchmark-action/github-action-benchmark@v1`](https://github.com/benchmark-action/github-action-benchmark).
The CI workflow `.github/workflows/benchmark.yml` on `main` writes one
new datapoint per merge into `perf/bench/data.js`. PR runs only
*compare* against this baseline; they do not push.

Do not edit by hand.
